# Flussdiagramm-Spiel für iOS

Native SwiftUI-Version des Lernspiels für iPhone und iPad.

## Projekt öffnen

1. Auf einem Mac [XcodeGen](https://github.com/yonaskolb/XcodeGen) installieren.
2. Im Ordner `ios` den Befehl `xcodegen generate` ausführen.
3. `FlussdiagrammSpiel.xcodeproj` in Xcode öffnen und starten.

Die App verwendet SwiftUI und benötigt iOS 17 oder neuer.
